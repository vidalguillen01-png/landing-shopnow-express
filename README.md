
🎨 ¿Por qué estos colores? - Explicación del diseño
🔵 Azul oscuro (#002060) - Confianza y seguridad
Elegí el azul oscuro como color principal porque:

✅ Transmite confianza - Los clientes asocian el azul con seguridad y profesionalismo.

✅ Da seriedad - Ideal para una tienda online que maneja compras y pagos.

✅ Genera calma - El azul es un color que relaja y da sensación de estabilidad.

✅ Contraste perfecto - Con el blanco crea una lectura clara y cómoda.

🧠 Psicología del color: El azul es el color más utilizado por empresas tecnológicas y financieras (Facebook, LinkedIn, PayPal). Transmite profesionalismo y fiabilidad.

🟠 Naranja (#F4A259) - Acción y energía
Elegí el naranja como color de acento porque:

✅ Llamar a la acción - El naranja atrae la atención y anima a hacer clic.

✅ Transmite energía - Da una sensación de dinamismo y urgencia.

✅ Genera entusiasmo - Es un color cálido que invita a comprar.

✅ Contraste con el azul - Son colores complementarios que crean un equilibrio visual atractivo.

🧠 Psicología del color: El naranja estimula el apetito y la acción. Por eso muchas tiendas online y restaurantes lo usan (Amazon, Uber, Fanta).

⚪ Blanco (#FFFFFF) - Claridad y minimalismo
Elegí el blanco como fondo porque:

✅ Legibilidad máxima - El texto sobre fondo blanco es más fácil de leer.

✅ Sensación de limpieza - Da un aspecto moderno y ordenado.

✅ No distrae - El blanco permite que los colores principales (azul y naranja) resalten.

⚪ Gris claro (#F5F8FA) - Separación visual
Elegí el gris claro para fondos de secciones porque:

✅ Crea jerarquía - Separa visualmente las diferentes secciones.

✅ Suaviza el diseño - Evita que la página sea demasiado "plana" y blanca.

✅ Estilo moderno - Es un color muy utilizado en diseño web actual.

📱 ¿Por qué el diseño responsive?
Mobile First - Prioridad en móviles
Diseñé la página pensando primero en celulares porque:

✅ El 80% del tráfico web viene de dispositivos móviles

✅ Los usuarios hacen compras desde el teléfono

✅ Google posiciona mejor las páginas "mobile-friendly"

Menú hamburguesa (☰)
En pantallas pequeñas, el menú se colapsa en un botón hamburguesa porque:

✅ Ahorra espacio - No ocupa la pantalla del celular

✅ Es familiar - Los usuarios reconocen este patrón de navegación

✅ Mejora la experiencia - Evita que el menú se vea apretado en móvil

🔧 ¿Por qué un formulario con validación?
El formulario de registro tiene validación porque:

✅ Evita errores - Asegura que los datos sean correctos antes de enviar

✅ Protege al usuario - La contraseña debe ser segura (8 caracteres + mayúscula)

✅ Mejora la experiencia - El usuario sabe rápidamente si cometió un error
